SAHS TV Display Android App

This project opens your SAHS TV Display page in a full-screen Android WebView.
URL:
https://www.swatacademic.com/admin/tv-display-public.php?key=SAHS2026TV

How to build APK:
1. Install Android Studio.
2. Open this folder: SAHS_TV_Display_App
3. Wait for Gradle Sync to finish.
4. Click Build > Build Bundle(s) / APK(s) > Build APK(s).
5. Android Studio will create app-debug.apk.
6. Copy APK to USB and install it on your TV box.

TV box notes:
- Allow Install from Unknown Sources.
- Open the app once after installation.
- Boot auto-start may work on many Android boxes, but some TV boxes block auto-start. In that case use an Auto Start app or set it from device settings.

Features:
- Full-screen display
- Hides Android navigation controls as much as possible
- Opens your website directly
- Auto-refresh every 3 minutes
- Landscape orientation
- Back button disabled
